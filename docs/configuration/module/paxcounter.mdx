Модуль Подсчёт Посетителей считает количество людей, проходящих мимо, сканируя MAC‑адреса Wi‑Fi и BLE. Часто используется в магазинах, музеях и других публичных местах для мониторинга трафика и анализа данных.

Для работы модуля используйте прошивку версии 2.2.17 или выше.

:::note заметка
Модуль работает только на ESP32. Для работы Подсчёт Посетителей нужно выключить Wi‑Fi и Bluetooth в Settings > Network и Bluetooth.
:::

## Параметры модуля Подсчёт Посетителей {#paxcounter-module-config-values}

### Включено {#enabled}

Включает/выключает модуль.

### Интервал обновления {#update-interval}

Интервал (сек.) между отправками в Mesh при изменении состояния.

#### CLI

:::note заметка

Все параметры модуля Подсчёт Посетителей доступны в Python CLI версии 2.2.16 и выше.

:::

Примеры команд:

```shell title="Включить/выключить модуль Подсчёт Посетителей"
meshtastic --set paxcounter.enabled true
meshtastic --set paxcounter.enabled false
```

```shell title="Задать минимальный интервал отправки 900 секунд"
meshtastic --set paxcounter.paxcounter_update_interval 900
```

```shell title="Показать конфиг модуля Подсчёт Посетителей"
meshtastic --get paxcounter
```
