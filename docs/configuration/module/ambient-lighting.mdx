Модуль Окружающее освещение позволяет управлять встроенными LED и настраивать яркость и цвета. Изначально создан для RAK14001 RGB LED на NCP5623 по I2C. Параметры: LED State, Current, Red, Green, Blue.

Для работы модуля требуется Firmware версии 2.2.5 или новее.

## Параметры Окружающее освещение {#ambient-lighting-config-values}

### LED State {#led-state}

Включить/выключить LED.

### Current {#current}

Ток выхода LED. По умолчанию 10.

### Red {#red}

Уровень красного. Диапазон 0–255.

### Green {#green}

Уровень зеленого. Диапазон 0–255.

### Blue {#blue}

Уровень синего. Диапазон 0–255.

#### CLI

:::note заметка

Все параметры модуля Окружающее освещение доступны в Python CLI версии 2.2.3 и новее.

:::

Примеры команд:

```shell title="Включить/выключить LED"
meshtastic --set ambient_lighting.led_state 1
meshtastic --set ambient_lighting.led_state 0
```

```shell title="Установить ток LED: 5"
meshtastic --set ambient_lighting.current 5
```

```shell title="Установить яркость красного LED: 103"
meshtastic --set ambient_lighting.red 103
```

```shell title="Установить яркость зеленого: 234"
meshtastic --set ambient_lighting.green 234
```

```shell title="Установить яркость синего LED: 148"
meshtastic --set ambient_lighting.blue 148
```

```shell title="Получить конфиг модуля Окружающее освещение"
meshtastic --get ambient_lighting
```
