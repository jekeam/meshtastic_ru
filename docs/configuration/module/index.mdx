Модули входят в Firmware и расширяют возможности вашего Mesh или устройства.

|                                    Название                                    |                                                                                          Описание                                                                                           |
| :----------------------------------------------------------------------------: | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------: |
|       [Подсветка](/docs/configuration/module/ambient-lighting)                 |                                                       Регулировка яркости RGB‑светодиодов NCP5623 по I2C.                                                                                   |
|                 [Аудио](/docs/configuration/module/audio)                      |                                                       Включить поддержку голосовой связи Codec2 на некоторых устройствах.                                                                   |
|        [Шаблонные сообщения](/docs/configuration/module/canned-message)        |                                   Задайте набор шаблонных сообщений для отправки прямо с устройства с помощью, например, энкодера.                                                         |
|      [Датчик обнаружения](/docs/configuration/module/detection-sensor)         |                                         Настройка GPIO‑пина для мониторинга заданного уровня (high/low) и отправки текстовых уведомлений.                                                 |
| [Внешние уведомления](/docs/configuration/module/external-notification)         |                                        Входящие сообщения могут оповещать через внешние цепи, подключённые к устройству (LED, buzzer и т.п.).                                              |
|                  [MQTT](/docs/configuration/module/mqtt)                       |                     Пересылать пакеты на MQTT‑сервер. Позволяет пользователям локального Mesh общаться с пользователями другого Mesh через интернет.                                       |
|          [Информация о соседях](/docs/configuration/module/neighbor-info)      |                                                                 Отправлять в Mesh информацию о соседях 0‑hop.                                                                              |
|            [Paxcounter](/docs/configuration/module/paxcounter)                 |                                                       Считать количество устройств BLE и Wi‑Fi, проходящих мимо Node.                                                                      |
|            [Тест дальности](/docs/configuration/module/range-test)             | Отправлять сообщения с координатами GPS с заданным интервалом, чтобы проверить дальность связи ваших устройств. Требуется минимум одно устройство как отправитель и одно как приёмник. Приёмник(и) записывают все входящие сообщения в CSV. |
|       [Удалённое железо](/docs/configuration/module/remote-hardware)           |                                                           Удалённо устанавливать и читать состояние GPIO через Mesh.                                                                       |
|            [Serial](/docs/configuration/module/serial)                         |                                                             Отправлять сообщения по Mesh, передавая строки через Serial‑порт.                                                              |
|  [Хранение и пересылка](/docs/configuration/module/store-and-forward-module)   |                                                 Хранит сообщения на устройстве для доставки после повторного подключения клиентов к Mesh.                                                  |
|             [Телеметрия](/docs/configuration/module/telemetry)                 |                                                   Подключайте датчики к устройству и передавайте показания в Mesh с заданным интервалом.                                                  |
|            [Traceroute](/docs/configuration/module/traceroute)                 |                                                     Отслеживать, какие Node задействованы при доставке сообщения до назначения.                                                            |
