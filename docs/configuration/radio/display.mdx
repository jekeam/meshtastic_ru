Параметры дисплея: время работы экрана, интервал авто‑карусели, всегда показывать север, формат GPS, единицы измерения, перевернуть экран, тип OLED, режим дисплея, жирный заголовок, пробуждение по касанию/движению, ориентация компаса и 12‑часовой формат. Настройки дисплея отправляются через админ‑сообщение `Config.Display` protobuf.

## Параметры дисплея {#display-config-values}

### Время работы экрана {#screen-on-duration}

Как долго экран остаётся включён после нажатия кнопки или получения сообщений.

### Интервал авто‑карусели {#auto-carousel-interval}

Автоматически переключает страницы на экране с указанным интервалом.

### Всегда показывать север {#always-point-north}

Если включено, индикатор курса вне круга всегда указывает на север. По умолчанию выключено: верх экрана — это ваше направление, индикатор севера перемещается по кругу.

### Формат GPS {#gps-format}

Формат отображения координат на экране устройства.

Допустимые значения:

| Значение |            Описание             |
| :------: | :-----------------------------: |
|  `DEC`   |       Десятичные градусы        |
|  `DMS`   |   Градусы, минуты, секунды      |
|  `UTM`   | Universal Transverse Mercator   |
| `MGRS`   | Military Grid Reference System  |
|  `OLC`   | Open Location Code (Plus Codes) |
| `OSGR`   | Ordnance Survey Grid Reference  |
|  `MLS`   | Maidenhead Locator System       |

### Единицы измерения {#preferred-display-units}

Переключение между `METRIC` (по умолчанию) и `IMPERIAL`.

### Перевернуть экран {#flip-screen}

Если включено, экран поворачивается на 180° для случаев, когда дисплей установлен вверх ногами.

### Тип OLED‑контроллера {#oled-definition}

Тип контроллера OLED по умолчанию определяется автоматически, но можно указать вручную, если автоопределение не сработало. Для SH1107 предполагается квадратный дисплей 128×128, как GME128128-1.

Допустимые значения:

|     Значение     |                 Описание                  |
| :--------------: | :---------------------------------------: |
|   `OLED_AUTO`    |     Автоопределение контроллера дисплея   |
| `OLED_SSD1306`   |         Всегда использовать SSD1306       |
|  `OLED_SH1106`   |          Всегда использовать SH1106       |
|  `OLED_SH1107`   | Всегда использовать SH1107 (128×128)      |

### Режим дисплея {#display-mode}

Режим можно установить в `DEFAULT` (по умолчанию), `TWOCOLOR`, `INVERTED` или `COLOR`. `TWOCOLOR` предназначен для OLED с первой строкой другого цвета. `INVERTED` инвертирует эту двухцветную зону, делая заголовок с белым фоном на монохромных дисплеях.

### Жирный заголовок {#heading-bold}

В режимах `INVERTED` и `TWOCOLOR` заголовок может быть плохо читаем. Этот параметр делает заголовок жирным для лучшей читаемости.

### Пробуждение по касанию/движению {#wake-on-tap-or-motion}

Включает пробуждение экрана при обнаружении движения (легкий удар по устройству) через подключённый акселерометр или ёмкостную кнопку.

### Ориентация компаса {#compass-orientation}

Как поворачивать или инвертировать компас на экране.

### 12‑часовой формат времени {#use-12-hour-clock}

Если `false`, время на экране в 24‑часовом формате. Если `true`, в 12‑часовом.

#### CLI

:::note заметка

Все параметры доступны в Python CLI. Примеры ниже:

:::

| Параметр                         | Допустимые значения                                        | По умолчанию                 |
| -------------------------------- | ---------------------------------------------------------- | ---------------------------- |
| display.auto_screen_carousel_secs | `integer`                                                  | Значение `0` — выкл.         |
| display.compass_north_top         | `false`, `true`                                            | `false`                      |
| display.flip_screen               | `false`, `true`                                            | `false`                      |
| display.gps_format                | `DEC`, `DMS`, `UTM`, `MGRS`, `OLC`, `OSGR`, `MLS`          | `DEC`                        |
| display.oled                      | `OLED_AUTO`, `OLED_SSD1306`, `OLED_SH1106`, `OLED_SH1107`  | `OLED_AUTO`                  |
| display.screen_on_secs            | `integer`                                                  | Значение `0` — 10 минут.     |
| display.units                     | `METRIC`, `IMPERIAL`                                       | `METRIC`                     |
| display.displaymode               | `DEFAULT`, `TWOCOLOR`, `INVERTED`, `COLOR`                 | `DEFAULT`                    |
| display.heading_bold              | `false`, `true`                                            | `false`                      |
| display.wake_on_tap_or_motion     | `false`, `true`                                            | `false`                      |

:::tip подсказка

Так как устройство перезагружается после каждой команды в CLI, при настройке нескольких значений удобнее объединять их в одну команду.

```shell title="Пример:"
meshtastic --set display.screen_on_secs 120 --set display.gps_format UTM
```

:::

```shell title="Время работы экрана (значение 0 — 10 минут)"
meshtastic --set display.screen_on_secs 0
meshtastic --set display.screen_on_secs 120
```

```shell title="Интервал авто‑карусели (0 — выкл.)"
meshtastic --set display.auto_screen_carousel_secs 0
// Set to 2 Minutes (120 Seconds)
meshtastic --set display.auto_screen_carousel_secs 120
```

```shell title="Формат GPS на экране устройства"
meshtastic --set display.gps_format UTM
```
