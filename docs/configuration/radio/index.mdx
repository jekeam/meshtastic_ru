В прошивке Meshtastic есть несколько разделов конфигурации — их выделили отдельно, чтобы отправлять как небольшие админ‑сообщения по Mesh.

|         Раздел          |                                                                                                   Описание                                                                                                   |
| :---------------------: | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------: |
| [Bluetooth](./bluetooth) |                                                              Параметры Bluetooth: Включено, режим привязки и фиксированный PIN.                                                                              |
|  [Channels](./channels)  |                                                                 Параметры Channel: индекс, роль и настройки.                                                                 |
|    [Device](./device)    |                                                  Параметры устройства: роль устройства, вывод в Serial, отладочный лог и сброс к заводским.                                                   |
|   [Display](./display)   |                                     Параметры дисплея: время включенного экрана, интервал автокарусели, «всегда на север» и формат GPS.                                     |
|      [LoRa](./lora)      | Параметры LoRa: регион, профиль модема, макс. переходов, мощность передачи, полоса, Spread Factor (SF), Coding Rate (CR), смещение частоты, отключить передачу и Ignore Incoming Array. |
|   [Network](./network)   |                                                  Параметры сети: включить WiFi, WiFi SSID, WiFi PSK, режим WiFi и NTP‑сервер.                                                   |
|  [Position](./position)  |                       Параметры позиции: включить GPS, интервал обновления GPS, время попытки GPS, фиксированная позиция, Smart Broadcast, интервал вещания и флаги пакета позиции.                       |
|     [Power](./power)     |     Параметры питания: ток зарядки, энергосбережение, выключение после потери питания, переопределить множитель ADC, интервал ожидания Bluetooth, интервал Light Sleep и мин. интервал бодрствования.      |
|  [Security](./security)  |                                  Параметры безопасности: публичный ключ, приватный ключ, Admin Key, Is Managed, консоль Serial, отладочные логи и Admin Channel.                                   |
|      [User](./user)      |                                                             Параметры пользователя: полное имя, короткое имя и есть лицензия.                                                             |
