## Диапазоны частот Европы {#europe-frequency-bands}

Страны ЕС обычно подпадают под Директиву по радиооборудованию ([2014/53/EU](http://data.europa.eu/eli/dir/2014/53/2023-10-01)) и её поправки, внедряемые в местные законы. Полезная информация есть [здесь](https://single-market-economy.ec.europa.eu/single-market/european-standards/harmonised-standards/radio-equipment_en).
Ниже выбраны диапазоны с учётом доступной мощности и duty cycle. Можно использовать и другие частоты для специфичных задач, но они пока не поддерживаются «из коробки».

### 433 МГц {#433-mhz}

Максимальная мощность для Европы: +10 dBm ERP ([эффективная излучаемая мощность](https://en.wikipedia.org/wiki/Effective_radiated_power)).

Диапазон: 433–434 МГц.

Определены четыре частотных слота для стандартной преднастройки `LongFast`. После сброса до заводских будет выбран слот 4 с центральной частотой 433.875 МГц.

### 868 МГц {#868-mhz}

Максимальная мощность для Европы: +27 dBm ERP ([эффективная излучаемая мощность](https://en.wikipedia.org/wiki/Effective_radiated_power)).

Диапазон: 869.40–869.65 МГц. Это уже, чем 863–870 МГц, определённые как диапазон SRD ([Short Range Devices](https://www.etsi.org/technologies/short-range-devices)), но позволяет выше ERP и duty cycle 10%.

Определён один частотный слот для стандартной преднастройки `LongFast`. После сброса до заводских будет выбран слот 1 с центральной частотой 869.525 МГц.

Отметим, что 868 МГц — самый популярный диапазон для Meshtastic в Европе.

## Диапазоны частот Северной Америки {#north-america-frequency-bands}

### 915 МГц (ISM) {#915-mhz-ism-band}

Максимальная выходная мощность для Северной Америки: +30 dBm ERP ([эффективная излучаемая мощность](https://en.wikipedia.org/wiki/Effective_radiated_power)).

Диапазон: 902–928 МГц.

В Северной Америке доступные частотные слоты зависят от настройки полосы (влияет выбранная преднастройка радио). Стандартная преднастройка LongFast даёт 104 частотных слота. После сброса до заводских по умолчанию выбирается LongFast со слотoм 0, что по алгоритму хеширования канала даёт слот 20 (центр 906.875 МГц).

## Скорости передачи {#data-rates}

### Что учитывать {#considerations}

Доступны разные варианты скорости при выборе частотного слота; они обратно пропорциональны теоретической дальности.

- Spreading Factor (SF) — насколько «растягиваем» данные во времени.
  - Каждый шаг SF удваивает время передачи.
  - Каждый шаг SF добавляет примерно 2.5 дБ к линк-бюджету.
- Bandwidth — какая ширина спектра используется.
  - Удвоение полосы отнимает почти 3 дБ линк-бюджета.
  - Полоса меньше 31 может быть нестабильной без качественного кварцевого генератора.
- Coding Rate — сколько избыточности для устойчивости к шуму.
  - Увеличение кодирования повышает надёжность и снижает скорость.
  - 4/5 — 1.25x накладные расходы
  - 4/6 — 1.5x накладные расходы
  - 4/7 — 1.75x накладные расходы
  - 4/8 — 2x накладные расходы

### Кастомные настройки {#custom-settings}

Кастомные настройки можно задать через [поддерживаемый софт](/docs/software).

После применения настроек нужно перезапустить устройство. После перезапуска сгенерируется новый ключ, и вам нужно будет поделиться новым QR[^2]-кодом или URL со всеми остальными устройствами.

## Криптография {#cryptography}

Предварительно общий ключ (PSK) может быть последовательностью AES128 или AES256. Также шифрование можно отключить — это может быть полезно при работе по радиолюбительской лицензии.

[^1]: «Ближняя Турбо» — самая быстрая преднастройка и единственная с полосой 500 кГц. Из‑за широкой полосы разрешена не во всех регионах.
