## Прошить устройство

:::info
Теперь можно использовать [Meshtastic Web Flasher](https://flasher.meshtastic.org), чтобы скачать и скопировать прошивку на устройства nRF52 или RP2040. Или следуйте инструкциям ниже, чтобы скачать и установить прошивку.
:::

Если вы только что сделали [Factory Erase](/docs/getting-started/flashing-firmware/nrf52/nrf52-erase/), устройство может автоматически запуститься в режиме загрузчика — тогда переходите сразу к шагу "Скопируйте подходящий файл прошивки..." для вашего девайса:

### nRF52

1. Скачайте и распакуйте свежую прошивку с [Meshtastic Downloads](https://meshtastic.org/downloads).
2. Подключите устройство к компьютеру USB-кабелем с передачей данных.
3. Дважды нажмите кнопку reset на устройстве (оно перейдёт в режим загрузчика).
4. На компьютере появится новый диск (Windows, Mac, Linux или Android).
5. Откройте этот диск — там должны быть три файла: `CURRENT.UF2`, `INDEX.HTM` и `INFO_UF2.TXT`.
6. Скопируйте подходящий файл прошивки (`firmware-DEVICE_NAME-X.X.X-xxxxxxx.uf2`) из релиза на этот диск.
7. Когда копирование завершится, устройство перезагрузится и установит прошивку Meshtastic.

### RP2040
1. Скачайте и распакуйте свежую прошивку с [Meshtastic Downloads](https://meshtastic.org/downloads).
2. Зажмите кнопку BOOTSEL и, не отпуская, подключите устройство к компьютеру по USB.
3. На компьютере появится новый диск (Windows, Mac, Linux или Android).
4. Откройте этот диск — там должны быть два файла: `INDEX.HTM` и `INFO_UF2.TXT`.
5. Скопируйте подходящий файл прошивки (`firmware-DEVICE_NAME-X.X.X-xxxxxxx.uf2`) из релиза на этот диск.
6. Когда копирование завершится, устройство перезагрузится и установит прошивку Meshtastic.

### Возможные нюансы прошивки

:::info

Перед прошивкой убедитесь, что у вас [RAK4631](https://docs.rakwireless.com/product-categories/wisblock/rak4631/overview), а не [RAK4631-R](https://docs.rakwireless.com/product-categories/wisblock/rak4631-r/overview). Если всё же RAK4631-R — не страшно. Железо одинаковое, просто нужно сменить загрузчик. Инструкция — [здесь](/docs/getting-started/flashing-firmware/nrf52/convert-rak4631r).

:::

:::info

Старые версии Meshtastic могли сохранять мусорные данные, из-за чего устройство уходит в crash loop при старте. Если при обновлении nRF52 с предыдущей версии возникли проблемы, может понадобиться полный factory reset внутренней флеш-памяти.

Перед прошивкой сначала выполните [factory erase для nRF52](/docs/getting-started/flashing-firmware/nrf52/nrf52-erase), затем снова [прошивайте](#flash-firmware).

:::
