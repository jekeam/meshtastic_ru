Устройства nRF52 от RAK принимают OTA‑обновления Firmware с телефона по Bluetooth. Старые загрузчики T‑Echo не поддерживают OTA.

:::caution осторожно

У OTA‑обновлений выше риск сбоя. Если процесс прервётся, устройство окажется в нерабочем состоянии и потребуется физический доступ для восстановления.

:::

#### Android

:::info информация
На момент написания текущая версия nRF DFU для Android (v2.3.0) несовместима с обновлениями Meshtastic Firmware. Используйте инструкции ниже для OTA через nRF Connect.
:::

OTA‑обновления для Android доступны в старой версии более продвинутого nRF Connect — **4.24.3**, её можно скачать на [GitHub Nordic Semiconductor](https://github.com/NordicSemiconductor/Android-nRF-Connect/releases/tag/v4.24.3).

1. Скачайте нужную версию Firmware со страницы [Meshtastic Download](/downloads) или с [Meshtastic GitHub](https://github.com/meshtastic/firmware/releases).
2. Распакуйте архив с Firmware.
3. Откройте nRF Connect и на вкладке SCANNER нажмите CONNECT напротив вашего устройства.
4. Если справа вверху написано DISCONNECT — переходите к шагу 5; если CONNECT — нажмите CONNECT.
5. Нажмите иконку DFU в правом верхнем углу.
6. Убедитесь, что выбран "Distribution Packet (ZIP)", и нажмите OK.
7. Выберите файл Firmware для вашего устройства (заканчивается на -ota.zip).
8. Обновление начнётся автоматически (это медленно).
9. После завершения устройство перезагрузится автоматически.

</TabItem>

#### Apple

OTA‑обновления доступны на iOS и iPadOS через приложение nRF Device Firmware Update из [Apple App Store](https://apps.apple.com/us/app/nrf-device-firmware-update/id1624454660)

1. Скачайте нужную версию Firmware со страницы [Meshtastic Download](/downloads), [Meshtastic GitHub](https://github.com/meshtastic/firmware/releases) или через приложение на iOS/iPadOS.
2. Распакуйте архив с Firmware.
3. Откройте nRF DFU и выберите файл Firmware для вашего устройства (заканчивается на -ota.zip).
4. Подключитесь к устройству.
5. Загрузите Firmware.

Автоблокировка iPhone может прервать загрузку Firmware по Bluetooth. Чтобы избежать этого, иногда касайтесь экрана или временно установите авто‑блокировку в "Never" на время загрузки, чтобы телефон не засыпал и процесс не прервался.

Если обновление не проходит, попробуйте настроить параметры пакетов:

1. В Settings включите "Packets Receipt Notification".
2. Уменьшите "Number of Packets". Некоторым помогло значение "5".
