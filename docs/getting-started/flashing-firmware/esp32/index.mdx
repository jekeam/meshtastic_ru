## Способы прошивки для устройств ESP32

1. [Web Flasher](/docs/getting-started/flashing-firmware/esp32/web-flasher.mdx) работает в браузерах Chrome и Edge и отлично подходит для быстрой прошивки устройств. Этот способ особенно рекомендуем новым пользователям — он простой и удобный.
2. [CLI Script](/docs/getting-started/flashing-firmware/esp32/cli-script) — «ручной» способ прошивки.
3. Прошивка через [внешний Serial-адаптер](/docs/getting-started/flashing-firmware/esp32/external-serial-adapter) — крайний вариант, если другие способы не сработали.

:::info информация

[веб-клиент](/docs/software/web-client) на meshtastic.local обновляется только при полном сбросе и переустановке устройства. Если выберете переустановку, получите последнюю (встроенную) версию веб-интерфейса. Чтобы сохранить настройки, [экспортируйте конфиг](/docs/software/python/cli#--export-config) перед переустановкой и загрузите его обратно после.

:::
