## Прошить устройство

1. Подключите устройство по USB
2. Откройте [flasher.meshtastic.org](https://flasher.meshtastic.org) _требуется браузер Chrome или Edge_
3. Следуйте инструкциям

### Подключить и настроить устройство

После прошивки Meshtastic на устройство можно перейти к первичной настройке.
