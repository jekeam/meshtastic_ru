## Настройка региона

Чтобы начать общаться в Mesh, укажите регион. Эта настройка определяет диапазон частот устройства и должна соответствовать вашему местоположению.

<TabItem value="android">

#### Android

1. Следуйте инструкциям по [установке](/docs/software/android/installation) и [использованию](/docs/software/android/usage/) для [Meshtastic Android](/docs/category/android-app).
2. Откройте приложение и подключитесь к устройству по USB Serial или Bluetooth.
3. После привязки нажмите "UNSET" рядом с именем устройства.
4. Выберите свой регион из списка.

</TabItem>
<TabItem value="apple">

#### Apple

:::info информация
Настройка Region, Modem Preset и Hop Limit доступна в iOS, iPadOS и macOS: Settings > Radio Configuration > LoRa.
:::

</TabItem>
<TabItem value="cli">

#### CLI

1. Установите [Meshtastic PythonCLI](/docs/software/python/cli/installation)
   ```sh
   pip3 install --upgrade pytap2
   pip3 install --upgrade meshtastic
   ```
2. Выполните команду, заменив `<REGION-CODE>` на код региона из списка выше.
   ```sh
   meshtastic --set lora.region <REGION-CODE>
   ```

</TabItem>
<TabItem value="web">

#### Web

1. Откройте Web-интерфейс Meshtastic: [client.meshtastic.org](https://client.meshtastic.org)
2. Перейдите в меню **LoRa**.
3. В разделе **Regional Settings** установите **Region**.
4. Нажмите **Save**.

</TabItem>
</Tabs>
