## Определите железо

:::note заметка

Этот гид предполагает, что вы уже купили устройства для Meshtastic. Если нет — посмотрите список [поддерживаемых устройств](/docs/hardware/devices/index.mdx).

:::

Перед началом важно понять, какое железо у вас. Meshtastic работает с устройствами на таких микроконтроллерах (MCU):

### ESP32

Чип ESP32 старее и потребляет больше энергии, чем nRF52, но поддерживает Wi‑Fi и Bluetooth. Поддерживаемые устройства на ESP32:

- LILYGO® TTGO T-Beam (>V1.1 рекомендуется)
- LILYGO® TTGO Lora (>V2.1 рекомендуется)
- Nano G1
- Station G1
- Heltec V3 и Wireless Stick Lite V3
- Модуль RAK11200 Core для модульных плат RAK WisBlock

### nRF52

Чип nRF52 заметно энергоэффективнее ESP32 и его проще обновлять, но он только с Bluetooth. Поддерживаемые устройства на nRF52:

- Модуль RAK4631 Core для модульных плат RAK WisBlock
- LILYGO® TTGO T-Echo

### RP2040

RP2040 — двухъядерный ARM-чип от Raspberry Pi. Поддерживаемые устройства на RP2040:

 - Raspberry Pi Pico + Waveshare LoRa Module (Примечание: **Bluetooth на Pico W пока не поддерживается в Meshtastic**)
 - Модуль RAK11310 Core для модульных плат RAK WisBlock

:::info информация

Если вашего устройства нет в списке, загляните в раздел [поддерживаемых устройств](/docs/hardware/devices/index.mdx), чтобы определить, какой MCU у вашего устройства, или задайте вопрос в нашем [Discord](https://discord.gg/ktMAKGBnBs).

:::

:::danger Опасно. Отключите кабель питания!

Никогда не включайте радио без подключённой антенны — так можно повредить радиочип!

:::

Перед подключением устройства Meshtastic к компьютеру выполните базовые проверки.

### Проверка кабеля {#verify-data-cable}

Некоторые кабели только для _зарядки_. Убедитесь, что ваш кабель умеет _передавать данные_. Подключите его к другому устройству (например, телефону) и попробуйте скопировать файл туда/обратно. Если передача файлов работает — кабель подходит, можно продолжать.

### Установить USB-драйвер

:::caution опасно

Устройствам nRF52/RP2040 обычно не нужны драйверы. Они используют загрузчик UF2 и определяются как флеш‑накопители. _Не_ скачивайте USB‑драйверы, если это не требуется для поддержки UF2.

:::

Если на вашем компьютере нужны драйверы, выберите вариант ниже и установите перед продолжением.
