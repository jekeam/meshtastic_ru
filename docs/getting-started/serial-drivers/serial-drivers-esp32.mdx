## Установить драйверы USB‑UART для ESP32

Может понадобиться драйвер Silicon Labs для [CP210X USB‑UART Bridge](https://www.silabs.com/products/development-tools/software/usb-to-uart-bridge-vcp-drivers)

Для некоторых новых плат нужен драйвер CH9102 (CH340/CH341).

#### Linux

- [CP210X USB‑UART Bridge — Скачать](https://www.silabs.com/products/development-tools/software/usb-to-uart-bridge-vcp-drivers)

#### macOS

- [CP210X USB‑UART Bridge — Скачать](https://www.silabs.com/products/development-tools/software/usb-to-uart-bridge-vcp-drivers)
- [Драйвер CH9102 — Скачать для macOS](https://github.com/WCHSoftGroup/ch34xser_macos)

</TabItem>

<TabItem value="windows">

#### Windows

- [CP210X USB‑UART Bridge — Скачать](https://www.silabs.com/products/development-tools/software/usb-to-uart-bridge-vcp-drivers)
- [Драйвер CH9102 — Скачать для Windows](http://www.wch.cn/downloads/CH343SER_ZIP.html)
- [Драйвер CH9102 — Скачать для Windows (прямая ссылка для Windows 7)](https://github.com/Xinyuan-LilyGO/CH9102_Driver)

</TabItem>

:::important

После установки драйвера перезагрузите компьютер, чтобы завершить установку.

На этом шаге при необходимости можно [проверить установку USB‑драйвера](/docs/getting-started/serial-drivers/test-serial-driver-installation).

:::

### Прошить устройство

После установки драйверов можно прошить устройство прошивкой Meshtastic. Выберите свой тип устройства.
