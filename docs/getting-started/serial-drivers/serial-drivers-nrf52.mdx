## Установить USB-драйверы nRF52/RP2040

:::caution опасно

Устройствам nRF52/RP2040 обычно не нужны Serial-драйверы. Они используют загрузчик UF2 и отображаются как флеш-накопители. НЕ скачивайте USB-драйверы, если только они не нужны для поддержки UF2.

:::

#### macOS

:::info информация

В последних версиях macOS USB-Serial драйвер встроен. Если вы уже ставили сторонние драйверы — удалите их.

:::

##### Удалить драйвер CH34x (macOS)

Если вы уже устанавливали драйверы WCH-IC CH340/CH341 для macOS («CH341SER_MAC») через CH34x_Install_V1.5.pkg, нужно удалить расширение ядра (kext):

1. Отключите устройство
2. Откройте Terminal и выполните:
3. `sudo rm -rf /Library/Extensions/usbserial.kext`
4. Перезагрузите компьютер

##### Установить драйвер CH34x

- [Драйвер CH34x — скачать для macOS](https://github.com/WCHSoftGroup/ch34xser_macos)

Также при необходимости можно [проверить установку Serial-драйвера](/docs/getting-started/serial-drivers/test-serial-driver-installation).

:::

### Прошить устройство

После установки драйверов можно прошить устройство прошивкой Meshtastic. Выберите тип вашего устройства.
