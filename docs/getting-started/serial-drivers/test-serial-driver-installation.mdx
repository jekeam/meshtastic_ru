### Проверка установки USB-драйвера

Можно проверить, что кабель — для данных (а не только зарядки), и что в системе установлены нужные драйверы, так:

#### Linux

1. Подключите устройство Meshtastic к USB-порту
2. Откройте Терминал и введите команду:

   ```shell
   lsusb
   ```

3. Должно быть что-то вроде:

   ```shell
   ID xxxx:xxxx Silicon Labs CP210x UART Bridge
   # or
   ID xxxx:xxxx QinHeng Electronics USB Single Serial
   # or
   ID xxxx:xxxx Adafruit WisCore RAK4631 Board
   ```

</TabItem>
<TabItem value="macos">

#### macOS

1. Откройте `Apple Menu  > About This Mac > More Info > System Report... > Hardware > USB`.
2. Должна быть одна из записей:

- `CP210X USB to UART Bridge Controller`
- `CH9102 USB to UART Bridge Controller`
- `WisCore RAK4631 Board`
- `USB Single Serial`

</TabItem>
<TabItem value="windows">

#### Windows

1. Откройте `Device Manager > Ports (COM & LPT)`
2. Должна быть одна из записей:

- `Silicon Labs CP210X USB to UART Bridge (COM5)`
- `Silicon Labs CH9102 USB to UART Bridge (COM5)`
- `USB-Enhanced-SERIAL CH9102 (COM5)`
- `USB Serial Device (COM5)`

</TabItem>
</Tabs>

:::info информация

Если устройство не видно:

- Убедитесь, что кабель не только для зарядки, но и для [передачи данных](/docs/getting-started#verify-data-cable).
- Возможно, нужно [переустановить USB-драйвер](/docs/getting-started/serial-drivers).

:::
